# 💊 Hora Certa

Aplicativo mobile desenvolvido em **Java no Android Studio** para auxiliar no gerenciamento de medicamentos pessoais, permitindo o controle de horários, lembretes e histórico de uso.

---

## 📱 Sobre o Projeto

O **Hora Certa** é um aplicativo mobile criado com foco em **Saúde Digital**, com o objetivo de ajudar usuários a manterem a adesão correta ao tratamento medicamentoso.

O aplicativo permite cadastrar medicamentos, receber lembretes e registrar o uso, contribuindo para uma rotina mais organizada e segura.

---

## 🎯 Funcionalidades

* ✅ Cadastro de paciente
* 💊 Cadastro de medicamentos
* ⏰ Lembretes de horários
* ✔ Registro de medicamentos tomados
* 📊 Histórico de uso
* 🔔 Notificações no dispositivo
* 🔐 Aceite de termos de uso (LGPD)

---

## 🧠 Tecnologias Utilizadas

* Java
* Android Studio
* SQLite (armazenamento local)
* XML (layouts)
* Android SDK

---


## 🔐 Segurança e LGPD

O aplicativo segue princípios básicos da LGPD:

* 📌 Coleta mínima de dados
* 📱 Armazenamento local (sem envio para servidores)
* 🔒 Dados não compartilhados com terceiros
* ☑️ Consentimento do usuário
* 🗑️ Possibilidade de exclusão de dados
